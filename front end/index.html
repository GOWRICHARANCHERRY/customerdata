📋 Excel Import

Import customer records from an Excel file. The Excel file should have the following columns:

  • Bill Number
  • Bill Date (YYYY-MM-DD format)
  • Customer Name
  • Phone Number
  • Address
  • Item Name
  • Item Type (Gold/Silver)
  • Item Amount
  • Interest (%)
  • Weight (grams)
  • Purity
  • Pending Money (number)
  • Extra Money Count (number)
  • Money Back Count (number)
  • Notes
  • Status (Active/Sold)
